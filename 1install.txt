1. MPTCP

	Install MPTCP using the instructions present in:
	https://multipath-tcp.org/pmwiki.php/Users/HowToInstallMPTCP?

	Dependencies for TCP connection:
	systemtap (sudo apt-get install systemtap or similar)

	Restart on MPTCP kernel, press shift on boot, go to Advanced Options and choose the mptcp-kernel

	sudo bash -c 'echo 1 > /proc/sys/net/ipv4/ip_forward' (enable packet forwarding)

2. Java (8 recommended)

	$ sudo add-apt-repository ppa:webupd8team/java
	$ sudo apt-get update
	$ sudo apt-get install oracle-java8-installer
